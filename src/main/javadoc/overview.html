JWhisper Common Library Overview

The jwhisper.common artifact provides the core cryptographic utilities, shared network protocols, and console input/output interfaces for the JWhisper ecosystem. It acts as the single source of truth for communication schemas, ensuring strict compatibility across nodes.

Key Structural Pillars

  • {@link io.github.artshp.jwhisper.common.protocol} Contains the canonical, positional Java Records (such as {@link io.github.artshp.jwhisper.common.protocol.RegisterRequest} and {@link io.github.artshp.jwhisper.common.protocol.EncryptedMessage}) that form the atomic, E2EE-compliant messaging framework.
  • {@link io.github.artshp.jwhisper.common.crypto} Exposes low-level Bouncy Castle wrappers for identity binding and robust key-derivation algorithms.
  • {@link io.github.artshp.jwhisper.common.io} Features {@link io.github.artshp.jwhisper.common.io.ConsoleUtils}, a specialized, ASCII-only user interaction utility incorporating input validation, try-catch retry loops, and zeroed-out secure password handling.
  • {@link io.github.artshp.jwhisper.common.exception} Defines application-wide custom checked exceptions (like {@link io.github.artshp.jwhisper.common.exception.InputRetryException}) for clean, safe system shutdowns.
